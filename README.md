# Heritage Horsepower

A premium, multi-page React website for a luxury automotive company.

## Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router DOM** - Navigation
- **Framer Motion** - Animations
- **Lucide React** - Icons

## Design Theme

- Dark luxury theme with charcoal/black backgrounds
- British racing green accents
- Gold/brass accent colors
- Large spacing and premium layout
- Smooth animations with Framer Motion
- High quality classic car imagery from Unsplash

## Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Features

### Pages
- **Home** - Full landing page with all sections
- **About** - Company story, values, and timeline
- **Vehicle Sourcing** - Sourcing process and categories
- **Automotive Experiences** - Curated driving experiences and events
- **Contact** - Contact form and location information

### Animations (Framer Motion)
- Fade in sections on scroll
- Parallax hero effect
- Card hover lift animations
- Button hover glow effects
- Smooth page transitions

### Components
- Responsive Navbar with mobile menu
- Premium Footer with contact info
- Reusable Button with multiple variants
- Card component with hover effects
- SectionHeading for consistent typography

## Design System

### Colors
- `heritage-black` (#0a0a0a) - Primary background
- `heritage-charcoal` (#141414) - Secondary background
- `heritage-gold` (#c9a961) - Primary accent
- `heritage-green` (#1b4d3e) - Secondary accent
- `heritage-cream` (#f5f0e6) - Primary text
- `heritage-gray-light` (#8a8a8a) - Secondary text

### Typography
- **Headings**: Playfair Display (serif)
- **Body**: Inter (sans-serif)

### Spacing
- Section padding: `py-24`
- Container max-width: `max-w-7xl`
- Card border-radius: `rounded-2xl`
